# utility_functions

#### Project Setup:
* Add npm:
  * `npm init -y`
* Add .gitignore file:
  * `touch .gitignore`
* Install jest:
  * `yarn add --dev jest`
* Install Babel:
  * `yarn add @babel/present-env --dev`
* Add `.babelrc` file with this content:
  * ```
     {
        presets: ['@babel/preset-env']
     }
    ```
* Add 'watch' command to yarn scripts:
   * `jest --watchAll`
